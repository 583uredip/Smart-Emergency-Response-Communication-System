🚨 Smart Emergency Response System

A C# .NET Windows Forms based emergency management system designed to improve communication between citizens, hospitals, police, and administrators in Bangladesh.

✨ Features
Accident, crime & medical emergency reporting
Blood donation request system
Admin dashboard for user management
Hospital & Police management modules
Secure login & user authentication
🛠️ Technologies
C#
.NET Framework
Windows Forms
SQL Database
👨‍💻 About This Project

This project was developed as our University Final Project by a team of two members. It was a challenging journey full of learning, problem-solving, and hard work. We successfully completed and submitted the project with dedication and passion.
